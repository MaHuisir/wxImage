# 朋友圈神器 - 智能图片压缩工具

一个现代化、高效能的图片压缩网站，专为朋友圈分享优化。

## ✨ 核心功能

### 图片压缩
- ✅ 支持 100MB 以内的图片上传
- ✅ 自动缩放长边至 ≤ 1080px
- ✅ 强制转换为 JPG 格式
- ✅ 最终文件大小 ≤ 2MB
- ✅ 智能质量调整（60% - 95%）

### 宽高比例
- ✅ **1:1 正方形** - 适合头像、方形图片
- ✅ **4:3 标准** - 适合风景、人物照片

### 批量处理
- ✅ 批量选择图片
- ✅ 批量压缩处理
- ✅ 批量下载文件
- ✅ 单个文件也支持

### 用户体验
- ✅ PC 和移动端完全适配
- ✅ 科技感蓝调设计
- ✅ 流畅的交互动画
- ✅ 实时压缩进度显示
- ✅ 详细的统计信息

## 🚀 快速开始

### 本地开发

```bash
# 无需安装依赖，直接打开
open index.html

# 或使用本地服务器
python -m http.server 8000
# 访问 http://localhost:8000
```

### 部署到 Netlify

#### 方式一：拖拽部署（最简单）
1. 访问 [Netlify Drop](https://app.netlify.com/drop)
2. 将整个项目文件夹拖拽到页面
3. 等待部署完成，获得公开链接

#### 方式二：Git 连接
1. 将项目推送到 GitHub
2. 在 Netlify 中连接 GitHub 仓库
3. 自动部署完成

#### 方式三：Netlify CLI
```bash
# 安装 Netlify CLI
npm install -g netlify-cli

# 登录
netlify login

# 部署
netlify deploy --prod
```

## 📊 压缩规则

|------|------|
| 长边 | ≤ 1080px |
| 文件大小 | ≤ 2MB |
| 格式 | JPG |
| 宽高比 | 1:1 或 4:3 |
| 质量 | 60% - 95% 可调 |

## 🔧 自定义配置

编辑 `index.html` 中的 CSS 变量来改变主题：

```css
:root {
    --primary: #0066ff;           /* 主色 */
    --secondary: #00d4ff;         /* 辅色 */
    --bg-dark: #0a0e27;           /* 背景 */
    --text-primary: #ffffff;      /* 文字 */
}
```

## 🎯 使用场景

1. **朋友圈分享** - 快速压缩照片，保持清晰度
2. **社交媒体** - 优化各平台的图片尺寸
3. **邮件附件** - 减小文件大小，加快上传

## 💡 使用建议

- **高质量照片**: 选择 85% - 95% 质量
- **快速分享**: 选择 60% - 75% 质量
- **风景照**: 使用 4:3 比例
- **人物照**: 使用 1:1 比例
