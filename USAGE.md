# 朋友圈神器 - 使用指南

## 🎯 快速开始

### 1. 打开网站
- 本地：在浏览器中打开 `index.html`
- 在线：访问部署后的 Netlify 链接

### 2. 上传图片
**三种方式选择：**
- 点击上传区域选择文件
- 拖拽图片到上传区域
- 支持多选，一次上传多张

### 3. 调整设置
在右侧设置面板中：
- **宽高比例**：选择 1:1（正方形）或 4:3（标准）
- **质量等级**：选择 60% / 75% / 85% / 95%

### 4. 压缩处理
- **单个压缩**：点击文件右侧的"⚡ 压缩"按钮
- **全部压缩**：点击"⚡ 全部压缩"按钮
- 实时显示压缩进度

### 5. 下载文件
- **单个下载**：点击文件右侧的"⬇️ 下载"按钮
- **全部下载**：点击"⬇️ 全部下载"按钮
- 文件自动保存为 JPG 格式

## 📊 功能详解

### 上传区域
```
📸 选择或拖拽图片
支持 JPG、PNG、WebP 等格式
单个文件 ≤ 100MB
```

### 压缩设置

#### 宽高比例
| 选项 | 用途 | 示例 |
|------|------|------|
| 1:1 正方形 | 头像、方形照片 | 400×400px |
| 4:3 标准 | 风景、人物照 | 1080×810px |

#### 质量等级
| 等级 | 质量 | 文件大小 | 用途 |
|------|------|---------|------|
| 低 (60%) | 较低 | 最小 | 快速分享 |
| 中 (75%) | 中等 | 中等 | 推荐使用 |
| 高 (85%) | 较高 | 较大 | 高质量需求 |
| 超高 (95%) | 最高 | 最大 | 专业用途 |

### 文件列表

每个文件显示：
- 📄 **文件名** - 原始文件名
- 📦 **原始大小** - 上传前的大小
- ✅ **压缩后** - 压缩后的大小（完成后显示）
- 🔄 **进度条** - 实时压缩进度
- 🏷️ **状态标签** - 处理中/已完成/失败

### 统计信息

顶部显示四个关键指标：
- **总文件数** - 已上传的图片数量
- **原始大小** - 所有图片的总大小
- **压缩后** - 所有压缩后的总大小
- **压缩率** - 平均压缩比例

## ⚙️ 压缩规则详解

### 尺寸调整
```
原始图片：2000×1500px
↓
缩放到长边 ≤ 1080px：1080×810px
↓
裁剪到 4:3 比例：1080×810px（已符合）
↓
最终：1080×810px
```

### 文件大小控制
```
压缩过程：
1. 按选定质量压缩（60%-95%）
2. 如果 > 2MB，自动降低质量
3. 重复直到 ≤ 2MB
4. 保证最终文件 ≤ 2MB
```

### 格式转换
```
支持输入：JPG、PNG、WebP、GIF 等
↓
统一输出：JPG 格式
↓
优点：兼容性好、文件小、加载快
```

## 🆘 常见问题

### Q: 为什么压缩后文件还是很大？
A: 
- 检查原始图片分辨率是否很高
- 尝试降低质量等级
- 某些图片类型（如高清照片）压缩空间有限

### Q: 支持哪些图片格式？
A: 支持所有常见格式：JPG、PNG、WebP、GIF、BMP 等

### Q: 压缩后会丢失质量吗？
A: 
- JPG 是有损压缩，会有轻微质量损失
- 但在 75% 质量下，肉眼难以察觉
- 可选择更高质量等级保留更多细节

### Q: 可以压缩视频吗？
A: 不支持，仅支持图片文件

### Q: 如何批量处理大量图片？
A: 
1. 一次上传多张（支持多选）
2. 调整设置
3. 点击"⚡ 全部压缩"
4. 点击"⬇️ 全部下载"

### Q: 下载的文件在哪里？
A: 
- 通常在浏览器的"下载"文件夹
- 可在浏览器设置中修改下载位置

## 🚀 高级技巧
